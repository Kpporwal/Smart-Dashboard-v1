# CryptoTracker Pro

A real-time, professional-grade cryptocurrency dashboard that tracks market trends, price fluctuations, and live data for top assets like Bitcoin, Ethereum, and Solana.

## 🚀 Features
- **Live Data:** Fetches real-time price data using the CoinGecko API.
- **Auto-Refresh:** Automatically updates prices every 30 seconds for up-to-the-minute tracking.
- **Interactive Charts:** Visual representation of 7-day market trends using Chart.js.
- **User Persistence:** Remembers your selected coin preference using LocalStorage.
- **Responsive Design:** A sleek, dark-themed UI built with Tailwind CSS that works on mobile and desktop.
- **Market Sentiment:** Instant indicator for positive or negative trends.

## 🛠 Tech Stack
- **Frontend:** HTML5, Tailwind CSS
- **Visualization:** Chart.js
- **Data Source:** [CoinGecko Public API](https://www.coingecko.com/en/api)
- **Automation:** JavaScript (setInterval & LocalStorage)
